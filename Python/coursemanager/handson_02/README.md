# Hands-On 2 – Django Models, ORM & Admin

## Topics Covered
- Django Models
- Model Relationships (ForeignKey)
- Migrations
- Django ORM (CRUD Operations)
- QuerySet Filtering
- Aggregation (annotate, Count)
- select_related()
- F() Expressions
- Django Admin Customization
- unique_together Constraint

## Commands Used

```bash
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
python manage.py shell
```

Successfully created models, performed CRUD operations using Django ORM, customized the Django Admin interface, and verified model constraints.
